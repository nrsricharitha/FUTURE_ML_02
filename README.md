Task Title:Support Ticket Classification & Prioritization 

This project builds a real-world NLP-based Machine Learning system that automatically:
reads customer support tickets (text)
classifies them into categories
predicts priority levels (High / Medium / Low)
This system helps companies reduce backlog, route tickets faster, and improve customer satisfaction.

 Project Overview:
Support teams often receive hundreds of customer tickets daily. Manually sorting them leads to:
delays in resolving urgent issues
inconsistent ticket categorization
wasted time and resources
This ML system solves that by using Natural Language Processing (NLP) and classification models to automate ticket routing.

Objectives:
1. Classify support tickets into categories
Examples:
Billing
Technical Issue
Account
General Query
2. Predict ticket priority
High
Medium
Low

📁 Dataset Used
Dataset Name:
Customer Support Ticket Dataset (Kaggle)
🔗 https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset
This dataset contains real support ticket text with labeled categories — perfect for multi-class NLP classification and priority prediction.
